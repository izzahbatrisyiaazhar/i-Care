# i-Care: A Web-Based Online Goods Donation Management System
=====================================================================================================
The system offers a structured management platform with a focus on emergency relief assistance. The 
users involved in the system are donors and administrators. There are a total of eight modules in 
the system. For donors, they could access the Register Module, Login Module, Make Goods Donation 
Module and Donation History Module. Meanwhile, administrators can access the Login Module, Manage 
Emergency Relief Program Module, Manage Donation Category Module, Manage Donation Activity Module 
and Report Module. 
======================================================================================================
## Requirements
======================================================================================================
* XAMPP v3.3.0 as local webserver that has a PHP Version 8.0.7
* PHP Language
* MySQL Database
* Google Chrome web browser (JavaScript enabled)
* Internet connection with a reasonable speed
======================================================================================================
## Prerequisites 
======================================================================================================
------------------------------------------------------------------------------------------------------
1.	Setting up a localhost server 
------------------------------------------------------------------------------------------------------
Step 1: Download XAMPP Software for Windows from https://www.apachefriends.org/download.html
Step 2: Install XAMPP Software in your device by follow the instruction given.
Step 3: Done 
------------------------------------------------------------------------------------------------------
2.	Download the source code zip file 
------------------------------------------------------------------------------------------------------
Step 1: Right-click on the zip file entitled “i-Care” on the shared Google Drive folder. 
Step 2: Click on “Download” 
Step 3: Done 

======================================================================================================
## Installation / Setup
======================================================================================================
Step 1: Open XAMPP Control Panel and 
Step 2: Start the “Apache” and “MySQL”.
Step 2: Extract the downloaded source code zip file.
Step 3: Copy the extracted source code folder 
Step 4: Paste the extracted source code folder into the XAMPP's "htdocs" directory 
Step 3: Browse the PHPMyAdmin in a web browser from (http://localhost/phpmyadmin) 
Step 4: Create a database naming, i-Care_db.
Step 5: Import the i-Care_db.sql file that is located inside the zip package in database folder.
Step 5: Browse the i-Care system in a browser from (http://localhost/i-Care/.) 
Step 6: Done
